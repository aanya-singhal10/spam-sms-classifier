# SMS Spam Detection

A Machine Learning project that classifies SMS messages as Spam or Ham (Not Spam).

## Overview

This project uses text preprocessing, TF-IDF vectorization, and machine learning algorithms to detect spam messages automatically.

The workflow includes:
- Data Cleaning
- Text Preprocessing
- Feature Engineering
- Model Training
- Model Evaluation
- Spam Prediction

## Technologies Used

- Python
- Pandas

- NLTK
- Scikit-learn
- Matplotlib
- Seaborn

## Features

- Text preprocessing using NLTK
- Tokenization
- Stopword removal
- Stemming
- TF-IDF Vectorization
- Machine Learning classification
- Spam prediction system
